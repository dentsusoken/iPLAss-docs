[[AiApplicationPolicyService]]
=== [.eeonly]#AiApplicationPolicyService#
This service manages application policy settings for AI features.

==== Interface Name
----
org.iplass.mtp.impl.ai.policy.AiApplicationPolicyService
----


==== Implementing Class Name
----
org.iplass.mtp.impl.ai.policy.AiApplicationPolicyService
----


==== Settings of AiApplicationPolicyService
Configure usage policies for AI features.

===== Configurable Items
[cols="1,1,3", options="header"]
|===
| Item | Value | Description
| aiChatAutoCompletionUserMessageMaxLength | int | Maximum length of the user message for AI chat auto-completion. The default value is 5000.
| aiChatAutoCompletionMaxConversationTurns | int | Maximum number of conversation history turns for AI chat auto-completion. The default value is 20.
| aiAutocompleteErrorClearTimeout | int | Timeout for clearing AI auto-complete errors (milliseconds). The default value is 5000.
|===

===== Example
[source,xml]
----
<service>
	<interface>org.iplass.mtp.impl.ai.policy.AiApplicationPolicyService</interface>

	<!-- Maximum length of the user message for AI chat auto-completion -->
	<property name="aiChatAutoCompletionUserMessageMaxLength" value="5000" />

	<!-- Maximum conversation history turns for AI chat auto-completion -->
	<property name="aiChatAutoCompletionMaxConversationTurns" value="20" />

	<!-- Timeout for clearing AI auto-complete errors (milliseconds) -->
	<property name="aiAutocompleteErrorClearTimeout" value="5000" />
</service>
----
