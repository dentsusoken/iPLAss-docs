[[VueJS_WebAPI_Layout]]
=== Top画面の作成
Top画面の構成は下図のようになっています。画面共通で利用するレイアウト用Vueコンポーネント `defaultLayout.vue` と `shippingLayout.vue` を用意し、Vue-routerを利用することでアクセスされたパスによって実際のコンテンツを切り替えています。
