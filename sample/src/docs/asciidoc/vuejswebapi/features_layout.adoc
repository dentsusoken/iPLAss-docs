=== Top画面の作成
* レイアウトの利用 +
Top画面の構成は下図のようになっています。画面共通で利用する2つのレイアウト用Vueコンポーネント `defaultLayout.vue` と `shippingLayout.vue` を用意し、Vue Routerを利用することでアクセスされたパスによって実際のコンテンツを切り替えています。
