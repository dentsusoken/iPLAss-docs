== EntityView
GEMはエンティティに対して、検索一覧と詳細画面を自動生成する仕組みを提供しています。
EntityViewを利用することで、エンティティの検索一覧と詳細画面をカスタマイズし、表示項目の制御や入力方法の変更、ユーザー（ロール）に合わせたレイアウトの設定を可能とします。
