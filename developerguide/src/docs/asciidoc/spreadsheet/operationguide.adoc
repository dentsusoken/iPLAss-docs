== 操作説明

=== SpreadSheet画面
SpreadSheet画面は、Entityのデータをスプレッドシート形式（グリッド）で編集・表示する画面です。
グリッド上で直接セルを編集したり、行の追加・削除、ファイル出力などを行えます。

.各部の説明

* **グリッド** ::
グリッド上にEntityのデータが行単位で表示されます。
`featureFlags` の設定に応じて、セルの編集、行の追加・削除、コピー＆ペースト、ソートなどが行えます。

* **列ヘッダー** ::
列ヘッダーをクリックすることでソートが可能です（`Feature Flags` の `allow sort` が有効な場合）。

* **フィルタ** ::
`filter items` に設定されたフィルタ項目を利用して、グリッドに表示するデータを絞り込めます。

=== セルの編集
グリッド上のセルをクリックすることで、そのセルが編集モードになります。
列に設定されたエディタタイプに応じて、適切な入力フォーム（テキストボックス、日付ピッカー、プルダウンなど）が表示されます。

編集を確定するには、`Enter` キーを押すか、別のセルを選択してください。
編集をキャンセルするには、`Esc` キーを押してください。

NOTE: 列定義の `editable` が無効、または `Feature Flags` の `allow add row` / `allow delete row` 等の設定によっては、編集操作が制限される場合があります。

=== 行の追加・削除
`Feature Flags` の設定が有効な場合、以下の操作が可能です。

* **行の追加** ::
`allow add row` が有効な場合、グリッド末尾に新規行を追加できます。

* **行の削除** ::
`allow delete row` が有効な場合、選択した行を削除できます。

=== コピー＆ペースト
`Feature Flags` の設定が有効な場合、以下の操作が可能です。

* **コピー** ::
`allow copy` が有効な場合、選択したセルまたは行のデータをコピーできます。

* **貼り付け** ::
`allow paste` が有効な場合、コピーしたデータをグリッドに貼り付けて一括入力できます。

=== ファイル出力
`Feature Flags` の `allow export` が有効な場合、グリッドのデータをファイルに出力できます。
出力形式は、設定で指定された形式（CSV、EXCEL）に従います。

=== 登録
グリッド上で編集したデータは、一括でEntityに登録されます。
登録処理には、`interrupter name` で指定したカスタム登録処理（`SpreadSheetRegistrationInterrupter`）が適用されます。
